Deploy The Worker

Free setup, simple steps.

  1. Install Node.js and create a Cloudflare account.
  2. Open the worker/ folder in this project.
  3. Run npm install.
  4. Run npx wrangler login.
  5. Change the Worker name in worker/wrangler.toml.
  6. Create a GitHub fine-grained personal access token for public API reads.
  7. Run npx wrangler secret put GITHUB_TOKEN and paste the token.
  8. Run npx wrangler deploy.
  9. Paste the resulting https://your-worker.workers.dev URL into the generator above.
